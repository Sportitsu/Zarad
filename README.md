## Usage

> app is now running online through :
[here](zara.herokuapp.com)

>to run the app locally:
```sh
ionic serve 
```
>to run the app on phone platforms
```sh
ionic serve --lab
```

## Running Test
-run front end test
```sh
npm run karma
```

-run back end test with coverage
```sh
npm test
```

-run front end cover test
```sh
npm run cover
```
## styling guide
```sh
gulp
```
